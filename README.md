# toggle-switch
Vue.jsを使用して、ライトモードとダークモードを切り替えるスイッチを作成  

## 環境

- npm v6.13.1
- VueCLI v3.8.2
- Vue v2.6.10  

## A switch to dark-mode

![dark mode](https://github.com/toshikisugiyama/darkmode-switch/blob/master/src/img/dark.png "dark mode")
